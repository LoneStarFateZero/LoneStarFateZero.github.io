说明

Spring数据键值框架使编写使用键值存储的Spring应用程序变得容易。

它最大程度地减少了与商店交互所需的冗余任务和样板代码。该框架非常适合Redis和Riak等键值存储。

在本文档中将介绍如何在默认的基于java.util.Map的实现中使用Spring Data Key Value 。
