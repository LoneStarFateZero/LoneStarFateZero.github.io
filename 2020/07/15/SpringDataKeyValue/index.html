SpringDataKeyValue

发表于 2020-07-15

[TOC]

说明

Spring数据键值框架使编写使用键值存储的Spring应用程序变得容易。

它最大程度地减少了与商店交互所需的冗余任务和样板代码。该框架非常适合Redis和Riak等键值存储。

在本文档中将介绍如何在默认的基于java.util.Map的实现中使用Spring Data Key Value 。

第三部分、自定义方法

Spring Data KeyValue按照方法名解析规则如下：

find+全局修饰+By+实体属性名称+限定词+连接词+（其他实体属性）+OrderBy+排序属性+排序方向

例如：

关键词规则样例
关键词 样例 JPQL样例
And findByNameAndAge(String name, Integer age) where x.name= ?1 and x.age= ?2
Or findByNameOrAge(String name, Integer age) where x.name= ?1 or x.age= ?2
Between findByAgeBetween(Integer age1, Integer age2) where x.age> ?1 and x.age< ?2
LessThan findByAgeLessThan(Integer age) where x.age < ?1
GreaterThan findByAgeGreaterThan(Integer age) where x.age > ?1
After findByDateAfer(Date date) where x.date > ?1
Before findByDateBefore(Date date) where x.date < ?1
IsNull findByAddressIsNull() where x.address is null
IsNotNull findByAddressIsNotNull() where x.address not null
Like findByNameLike(String name) where x.name like ?1
NotLike findByNameNotLike(String name) where x.name not like ?1
StartingWith findByNameStartingWith(String name) where x.name like ?1%
EndingWith findByNameEndingWith(String name) where x.name like %?1
Containing findByNameContaining(String name) where x.name like %?1%
OrderBy findByNameOrderByAge(String name) where x.name = ?1 order by x.age
Not findByNameNot(String name) where x.name <> ?1
In findByNameIn(Collection c) where x.name in ?1
NotIn findByNameNotIn(Collection c) where x.name not in ?1
TRUE findByActiveTrue() where x.active = 1
FALSE findByActiveFALSE() where x.active = 0

第四部分、分页&排序数据查询

1.分页查询

使用分页查询，需要自定义Repository接口继承PagingAndSortingRepository接口，从而使用其中的分页查询方法。

之后，便可构造Pageable对象，使用findAll(Pageable pageable)方法获取分页查询结果。

2.排序查询

同样，自定义Repository接口继承PagingAndSortingRepository接口，从而使用其中的排序查询方法。

第五部分、使用KeyValueTemplate
