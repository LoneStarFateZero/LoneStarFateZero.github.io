keys * 查看所有键 127.0.0.1:6379> keys *
1) “python”
2) “java”
3) “hello”
dbsize 查看键总数 127.0.0.1:6379> dbsize
(integer) 4
exists key 检查键是否存在 127.0.0.1:6379> exists exist_key
(integer) 1
127.0.0.1:6379> exists not_exist_key
(integer) 0
del key [key …] 删除键 127.0.0.1:6379> EXISTS exist_key
(integer) 1
127.0.0.1:6379> del exist_key
(integer) 1
expire key seconds 设置键过期 127.0.0.1:6379> set hello world
OK
127.0.0.1:6379> expire hello 10
(integer) 1
ttl key 查看键剩余过期时间
大于等于0的整数:键剩余的过期时间。
-1:键没设置过期时间。
-2:键不存在		 #还剩7秒
127.0.0.1:6379> ttl hello
(integer) 7
…
#还剩1秒
127.0.0.1:6379> ttl hello
(integer) 1
#返回结果为-2,说明键hello已经被删除
127.0.0.1:6379> ttl hello
(integer) -2
127.0.0.1:6379> get hello
(nil)

列表类型

特点：

  • 列表中的元素是有序的
  • 列表中的元素可以重复
  • 索引下标从左到右是0到N-1，从右到左是-1到-N

lpush key value [value …] 从左边插入元素
linsert key before|after pivot value 向某个元素前或后插入元素
